PlotControl

A PlotControl object consists of a Plot inside a scrolled pane and various controls in a panel. The Plot allows you to display many spectra. Plots wrapped by this object are given a name that is unique within an application (<plotn>) and are assigned an identifying number (0 upwards).

The controls in the panel allow you to:

  • apply independent scales in X and Y, thus zooming and scrolling the Plot,<li>
  • get a continuous readout of the cursor position,
  • display a vertical hair,<li>
  • select the current spectrum and see a list of those displayed,<li>
  • apply percentile cuts to the data limits.<li>

Methods are provided for scaling about the current centre, zooming in to a given region as well as just setting the X and Y scale factors.

You can also scroll to centre on a given X coordinate (or the one shown in the X coordinate readout field) and make the Plot fit itself to the current viewable height andor width.

A not-very exhaustive set of methods for querying the contents of the Plot (e.g. getting access to the plotted spectra) and adding and removing spectra from the Plot are provided.

Finally this object can make the Plot produce a postscript and JPEG output copy of itself.

See the Plot object for a description of the facilities it provides.

public void zoomToRectangularRegion(java.awt.geom.Rectangle2D region)

Zoomscroll the Plot to display a given rectangular region.

If the region has zero width or height then the zoom is merely incremented by 1 in both dimensions. Also if the scale factors are increased by more than a factor of 20, then they are clipped to this value (the most likely action here is a user accidently returning a very small region).

  • Parameter
    region
    the region to zoom into.
