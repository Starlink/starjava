SpectralFileFilter

A convenience implementation of java.io.FileFilter that filters out all files except for those type extensions that it knows about.

Extensions are of the type ".foo", which is typically found on Windows and Unix boxes, but not on Macinthosh. Case is ignored.

Example - create a new filter that filters out all files but gif and jpg image files:
SpectralFileFilter filter = new SpectralFileFilter(
new String{"gif", "jpg"}, "JPEG & GIF Images")

Constructors

public SpectralFileFilter()
Creates a file filter. If no filters are added, then all files are accepted.
  • See also
    • SpectralFileFilter.addExtension

public SpectralFileFilter(String extension)

Creates a file filter that accepts files with the given extension. Example: new SpectralFileFilter("jpg");
  • Parameter
    extension
    file extension.
  • See also
    • SpectralFileFilter.addExtension

public SpectralFileFilter(String[] filters)

Creates a file filter from the given string array. Example: new SpectralFileFilter( String {"gif", "jpg"} );

Note that the "." before the extension is not needed and will be ignored.

  • Parameter
    filters
    array of file extensions to use a filters.
  • See also
    • SpectralFileFilter.addExtension

Methods

public boolean accept(java.io.File pathname)
Tests whether or not the specified abstract pathname should be included in a pathname list.
  • Parameter
    pathname
    The abstract pathname to be tested
  • Returns true if and only if pathname should be included.

public void addExtension(String extension)

Adds a filetype "dot" extension to filter against.

For example: the following code will create a filter that filters out all files except those that end in ".jpg" and ".tif":
SpectralFileFilter filter = new SpectralFileFilter();
filter.addExtension("jpg");
filter.addExtension("tif");

Note that the "." before the extension is not needed and will be ignored.

  • Parameter
    extension
    a file extension to add.

public String getExtension(java.io.File f)

Return the extension portion of the file's name.
  • Parameter
    f
    a File object.
  • Returns the file extension.
  • See also
    • SpectralFileFilter.getExtension
    • java.io.FileFilter.accept
