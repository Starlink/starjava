RemoteUtilities

This class provides utility methods for remote control based access to SPLAT.

Methods

public static boolean isListening(Object[] contactDetails)
Find out if SPLAT is listening on the given hostname and port and recognises the given cookie.
  • Parameter
    contactDetails
    return from readContactFile.
  • Returns true if an instance of SPLAT was successfully contacted.

public static Object readContactFile()

Parse the contact file returning its contents as an Object array.
  • Returns array of three Objects. These are really the hostname String, an Integer with the port number and a String containing the validation cookie. Returns null if not available.

public static String sendRemoteCommand(Object[] contactDetails, String command)

Send a single beanshell command to a remote SPLAT beanshell interpreter. Needs the hostname, port and cookie from the contact file.
  • Parameters
    contactDetails
    return from readContactFile.
    command
    the beanshell command to execute.
  • Returns the string that returns from the command, null for an error.
  • Exceptions

    • Exception - exception resulting from any network problems.
