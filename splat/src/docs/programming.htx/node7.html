Package uk.ac.starlink.splat.data

MEMSpecDataImpl

This class provides an implementation of SpecDataImpl to access spectra stored in existing memory.
SpecData
SpecData defines an interface for general access to spectral datasets of differing fundamental data types and is the main data model used in SPLAT.
SpecDataFactory
This class creates and clones instances of SpecData that are appropriate to the data format of the given spectrum specification.
